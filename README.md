MedBot is an intelligent **medical chatbot** that leverages **LLaMA 2**, **LangChain**, and **Pinecone** to provide context‑aware answers from your medical PDFs. *(Scroll to the end of the Colab notebook to see the chat demo.)*

---

## 🚀 Features
- 📄 Load and process **medical PDFs** into knowledge chunks.  
- ✂️ Chunking via `RecursiveCharacterTextSplitter` *(chunk_size=500, overlap=20)*.  
- 🧠 Embeddings with **HuggingFace**: `sentence-transformers/all-MiniLM-L6-v2` *(384‑dim)*.  
- 🗂️ Vector DB: **Pinecone Serverless (AWS us-east-1)**, index **`medchat`** with **cosine** metric.  
- 🔎 Retrieval with LangChain’s Pinecone wrapper *(top‑k=2, chain_type="stuff")*.  
- 🤖 Answers generated by **LLaMA 2** via `CTransformers` *(max_new_tokens=512, temperature=0.8)*.

## 🧱 How It Works
1. **Load PDFs** with `DirectoryLoader(..., glob="*.pdf", loader_cls=PyPDFLoader)`  
2. **Split** into chunks *(500 / 20)* with `RecursiveCharacterTextSplitter`  
3. **Embed** using `HuggingFaceEmbeddings("sentence-transformers/all-MiniLM-L6-v2")`  
4. **Index** in Pinecone: creates/uses index **`medchat`** (dim **384**, metric **cosine**, serverless **AWS us-east-1**) and **upserts** vectors with `{ id, values, metadata: {"text": ...} }`  
5. **Prompting** with a safe QA prompt:
   ```text
   Use the following pieces of information to answer the user's question.
   If you don't know the answer, just say that you don't know...
   ```
6. **Retrieval‑QA** with LangChain:
   ```python
   qa = RetrievalQA.from_chain_type(
       llm=llm, chain_type="stuff",
       retriever=docsearch.as_retriever(search_kwargs={'k': 2}),
       return_source_documents=True,
       chain_type_kwargs={"prompt": PROMPT}
   )
   ```
7. **Chat** – Run the final cell to ask questions and view answers + sources.
